# CiberNotas del CiberCalvoCabrón

Repositorio de documentación y apuntes sobre ciberseguridad, pentesting y herramientas de análisis.

## Características

* Interfaz web ligera sin frameworks.
* Navegación mediante árbol jerárquico.
* Carga dinámica de contenido.
* Breadcrumb automático.
* Sistema de categorías visuales.
* Buscador con soporte para acentos y caracteres especiales.
* Etiquetas para clasificar contenido.
* Diseño orientado a documentación técnica offline.

Crear una base de conocimiento organizada y fácilmente navegable para centralizar conceptos, laboratorios, herramientas y procedimientos relacionados con la seguridad informática.
