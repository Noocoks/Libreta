Hydra

Herramienta de auditoría de autenticación para entornos autorizados.

pentesting credenciales diccionario ssh ftp

Hydra es una herramienta Open Source de auditoría de autenticación que permite evaluar la fortaleza de usuarios y contraseñas mediante pruebas automatizadas sobre múltiples servicios de red.
Su objetivo es detectar credenciales débiles y comprobar la eficacia de las políticas de seguridad en entornos autorizados, sin explotar vulnerabilidades del sistema ni modificar su configuración.


Casos de uso

  • Auditorías de seguridad sobre servicios de autenticación.
  • Detección de contraseñas débiles o reutilizadas.
  • Verificación de políticas de bloqueo y limitación de intentos.
  • Formación y laboratorios de ciberseguridad.
  • Pentesting autorizado.

Instalación

En distribuciones basadas en Debian, Ubuntu o Kali Linux puede instalarse directamente desde los repositorios oficiales. 

└─$ sudo apt update
└─$ sudo apt install hydra

Comprobación

└─$ which hydra
└─$ hydra -h
└─$ hydra -U

Si la instalación es correcta, el sistema localizará el ejecutable y mostrará la ayuda integrada junto con la lista de módulos disponibles.
