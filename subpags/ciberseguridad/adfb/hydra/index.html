Hydra

pentesting credenciales diccionario ssh ftp

Hydra es una herramienta de auditoría de seguridad utilizada en Ataques de fuerza bruta para evaluar la fortaleza de sistemas de autenticación mediante pruebas automatizadas de credenciales.

Su uso legítimo se limita a auditorías autorizadas y entornos de laboratorio controlados.


Casos de uso

  • Detectar credenciales débiles.
  • Evaluar la robustez de mecanismos de autenticación.
  • Ayudar a mejorar la seguridad antes de una intrusión real.

Servicios compatibles

  • SSH
  • FTP
  • HTTP
  • Otros protocolos soportados.

Instalación

└─$ sudo apt update
└─$ sudo apt install hydra
└─$ hydra -h
