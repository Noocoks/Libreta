Laboratorio - Hydra

Preparación del entorno

  • Máquina local o virtual (muy recomendable usar VM).
  • Red aislada o entorno de pruebas.

Contenedores Docker vulnerables

  • DVWA
  • Metasploitable
  • Otros entornos de prueba

Usuarios con contraseñas débiles

Ejemplos:

  • admin123
  • 12345678
  • paquito06
  • manolopiedra

Wordlists y ejecución en localhost

Crear archivos wordlist.txt y usuarios.txt con credenciales de prueba.

Ejemplo para wordlist.txt:

  • 123456
  • password
  • qwerty
  • admin123
  • letmein

Ejemplo para usuarios.txt:

  • admin
  • usuario1
  • pepe
  • maria
  • test

Ejecución de Hydra

Prueba de autenticación contra un servicio SSH en un entorno local y controlado. 

└─$ hydra -L usuarios.txt -P wordlist.txt ssh://192.168.1.100
