Medusa

Herramienta de auditoría de autenticación para entornos autorizados.


Medusa es una herramienta Open Source de auditoría de autenticación que permite evaluar la seguridad de distintos servicios de red mediante intentos automatizados de inicio de sesión.
Está orientada a comprobar la fortaleza de las credenciales y la eficacia de las políticas de autenticación en entornos autorizados, sin explotar vulnerabilidades ni modificar la configuración del sistema, limitándose únicamente a interactuar con los mecanismos de autenticación disponibles.


Casos de uso más frecuentes

Auditorías de seguridad

Permite comprobar la existencia de credenciales débiles y verificar que las políticas de autenticación implantadas protegen correctamente los servicios corporativos.


Laboratorios y formación

Es ampliamente utilizada en universidades, centros de formación y laboratorios de ciberseguridad para enseñar el funcionamiento de los mecanismos de autenticación y la importancia de aplicar buenas prácticas de seguridad.


Validación de políticas de seguridad

Permite verificar que mecanismos como el bloqueo de cuentas, la autenticación multifactor (MFA) o la limitación de intentos funcionan correctamente frente a intentos repetidos de autenticación.


Instalación

└─$ sudo apt update
└─$ sudo apt install medusa

Una vez finalizada la instalación, el sistema descargará automáticamente todas las dependencias necesarias.


Comprobación de la instalación

└─$ which medusa
└─$ medusa -V
└─$ medusa -h

Si han funcionado todos, ya tienes la herramienta disponible para su uso.
