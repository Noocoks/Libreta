ExifTool

Es una herramienta OSINT de línea de comandos muy conocida para leer, escribir y modificar metadatos en archivos multimedia y documentos.

Los objetivos principales de esta herramienta son:

  • Ver información oculta (fecha, cámara, GPS, etc.)
  • Modificar datos de archivos
  • Eliminar información privada
  • Automatizar organización de archivos
  • Apoyar análisis forense y verificación digital

Método de instalación:

└─$ sudo apt update && sudo apt upgrade -y

└─$ sudo apt install libimage-exiftool-perl

└─$ exiftool -ver

└─$ exiftool img.jpg
