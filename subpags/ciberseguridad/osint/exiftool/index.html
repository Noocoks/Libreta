ExifTool

osint metadatos forense gps imágenes

ExifTool es una herramienta de OSINT utilizada para leer, escribir y modificar metadatos en archivos multimedia y documentos.

Es una de las herramientas más utilizadas para análisis forense, investigación digital y auditorías relacionadas con información oculta en archivos.


Casos de uso

  • Ver información oculta (fecha, cámara, GPS, etc.).
  • Modificar metadatos de archivos.
  • Eliminar información privada antes de compartir documentos o imágenes.
  • Automatizar la organización de archivos.
  • Apoyar análisis forense y verificación digital.

Instalación


  sudo apt update && sudo apt upgrade -y

  sudo apt install libimage-exiftool-perl

  exiftool -ver

  exiftool imagen.jpg
