Holehe

osint correo cuentas reconocimiento usuarios

Holehe es una herramienta de OSINT diseñada para verificar si una dirección de correo electrónico está registrada en distintos servicios online.

Su objetivo principal es automatizar tareas de reconocimiento e investigación relacionadas con correos electrónicos y perfiles digitales.


Casos de uso

  • Comprobación de registros
    • Verificar si un correo electrónico existe en plataformas conocidas.
  • Investigación OSINT
    • Obtener información sobre la presencia digital de un objetivo.
  • Auditorías de seguridad
    • Detectar exposición de cuentas asociadas a un correo.
  • Gestión de identidad digital
    • Revisar servicios donde una dirección de correo está siendo utilizada.

Instalación


  └─$ sudo apt update && sudo apt upgrade -y

  └─$ sudo apt install python3 python3-pip git -y

  └─$ git clone https://github.com/megadose/holehe.git

  └─$ cd holehe/

  └─$ sudo python3 setup.py install

  └─$ holehe [correo@ejemplo.com](mailto:correo@ejemplo.com)

Ejemplo de uso


  └─$ holehe [usuario@dominio.com]

Holehe realizará comprobaciones automáticas en múltiples servicios compatibles y mostrará cuáles tienen una cuenta asociada al correo indicado.
