⚙️ Arquitectura - Gophish

  1. 1. Servidor web (Control Panel)

    Cuenta con una interfaz web donde diseñar campañas, correos, landing pages y perfiles objetivo.

  2. 2. Motor de envío de correos SMTP

    Permite el envío de correos simulados utilizando:

    • Servidores propios.
    • Servicios externos (entornos de laboratorio).
  3. 3. Landing Pages (páginas falsas de ataque)

    Simulan formularios de login u otras interacciones para medir el comportamiento de los usuarios.

  4. 4. Tracking y Logging

    Registra información como:

    • Aperturas de correos.
    • Clics en enlaces.
    • Envío de datos (credenciales o formularios).
