Arquitectura - Gophish

Gophish se estructura en varios componentes encargados de gestionar campañas, enviar correos, alojar páginas de captura y registrar la interacción de los usuarios.


Componentes

Servidor web

Proporciona la interfaz de administración utilizada para crear y gestionar campañas de phishing.

  • Plantillas de correo.
  • Landing pages.
  • Perfiles objetivo.
  • Estadísticas de campañas.

Motor SMTP

Encargado del envío de correos electrónicos simulados.

  • Servidores SMTP propios.
  • Servicios externos de laboratorio.

Landing Pages

Páginas web diseñadas para simular formularios de autenticación u otros escenarios de ingeniería social.


Tracking y Logging

Sistema encargado de recopilar métricas e interacciones.

  • Aperturas de correos.
  • Clics en enlaces.
  • Envío de formularios.
  • Captura de indicadores de compromiso.
