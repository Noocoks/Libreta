Landing Page - Código

Landing page utilizada durante las simulaciones de Gophish para capturar credenciales y registrar interacciones de usuarios en laboratorio.


Resumen

  • Interfaz corporativa simulada.
  • Formulario de autenticación.
  • Captura de credenciales mediante POST.
  • Indicador visual de carga tras el envío.

Notas

Formulario

Los campos username y password son procesados por Gophish cuando se habilita la captura de datos en la Landing Page.


Captura de credenciales

El formulario utiliza el método POST para enviar la información introducida por el usuario.


Spinner

Tras enviar el formulario se oculta la interfaz y se muestra una animación de carga para simular un proceso de validación.
