Código - Email Template

Plantilla HTML utilizada por Gophish para generar correos electrónicos de simulación.

Este código corresponde al contenido del correo enviado durante la campaña y puede contener variables dinámicas, enlaces de seguimiento y elementos visuales personalizados.


Función dentro de Gophish

  • Definir el aspecto visual del correo.
  • Personalizar mensajes para cada destinatario.
  • Insertar enlaces de seguimiento.
  • Simular comunicaciones corporativas.
  • Evaluar interacción de usuarios.

Notas

  • Compatible con el sistema de plantillas de Gophish.
  • Puede incluir variables dinámicas.
  • Se utiliza junto a una Landing Page.
  • Forma parte de una campaña de simulación controlada.
