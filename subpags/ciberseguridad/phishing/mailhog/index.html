MailHog

phishing smtp correo testing desarrollo

MailHog es una herramienta utilizada en Phishing y desarrollo para simular un servidor de correo electrónico.

Su función principal es capturar correos enviados por aplicaciones sin que estos sean enviados realmente a internet.


Funcionamiento

  • Simula un servidor SMTP para aplicaciones.
  • Recibe todos los correos enviados durante pruebas.
  • Permite visualizar mensajes desde una interfaz web.
  • Evita el envío accidental de correos reales.

Interfaz web

Normalmente se encuentra disponible en: 


  http://localhost:8025

Desde esta interfaz pueden revisarse mensajes, cabeceras, adjuntos y contenido HTML de los correos capturados.


Casos de uso

  • Pruebas de campañas de phishing en laboratorio.
  • Verificación de correos automáticos.
  • Pruebas de registro y recuperación de cuentas.
  • Desarrollo de aplicaciones con funcionalidades de correo.

/* Ideal para pruebas porque evita enviar correos reales mientras se desarrolla o audita una aplicación. */
