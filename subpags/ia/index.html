IA

En este apartado he ido creando poco a poco un mecanismo basado en inteligencia artificial para poder trabajar de la forma más rápida, eficiente y fácil posible. Para poder usar este sector de los apuntes primero tendremos que crear un entorno virtual con python con los siguientes comandos.

Antes de empezar, a continuación está la estructura recomendada para llevar a cabo el proyecto

~/ai/

├── venv/

├── .env

├── .gitignore

├── test_apis.py

Comenzamos creando una carpeta aparte con el entorno.

└─$ mkdir ~/ai && cd ~/ai

└─$ python3 -m venv venv

└─$ source venv/bin/activate

Con estas lineas de comando ya lo tendríamos hecho y estaremos ya dentro para empezar a trabajar.

❗Comando para entrar en el entorno directamente desde nuestro [~]:

└─$ cd ~/ai && source venv/bin/activate

.

Disclaimer: Puedes cambiar la ruta a tu gusto, pero te dejo la mía por motivos del tutorial.

Empezaremos creando el archivo con nuestras APIs

Guardaremos el contenido del archivo y le daremos estos permisos:

└─$ chmod 600 .env

De esta forma dejamos el archivo completamente privado a nuestro usuario, código y Github, totalmente para nuestro uso exclusivo.

Ahora solo queda hacer un pequeño archivo .py para poder probar que las claves están bien guardadas. Despues crearemos el .gitignore para decirle a Git que archivos NO debe subir al repositorio. Y solo falta acabar creando los archivos para usar OpenAI y Anthropic.

Con todo comprobado y realmente funcionando solo queda probar el mini programita que junta todo lo anterior, pero de forma algo más profesional. Juntar ambas IAs en el mismo código.
