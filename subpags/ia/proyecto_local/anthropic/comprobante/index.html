Comprobante Anthropic

Script básico para verificar que la API de Anthropic está correctamente configurada y accesible desde Python.

El objetivo es validar la carga de variables de entorno, la autenticación y la comunicación con el modelo.


Dependencias


  └─$ pip install anthropic python-dotenv

Código


  import os

  from dotenv import load_dotenv
  from anthropic import Anthropic

  load_dotenv()

  client = Anthropic(
      api_key=os.getenv("ANTHROPIC_API_KEY")
  )

  message = client.messages.create(
      model="claude-sonnet-4-0",
      max_tokens=100,
      messages=[
          {
              "role": "user",
              "content": "Hola"
          }
      ]
  )

  print(message.content[0].text)

Resultado esperado

  • La clave API es cargada correctamente.
  • La petición llega a Anthropic.
  • Claude devuelve una respuesta válida.
  • La respuesta aparece por terminal.
