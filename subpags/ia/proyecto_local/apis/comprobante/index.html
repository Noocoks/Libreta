Comprobación APIs

python dotenv variables-entorno openai anthropic api

Este script permite comprobar que las claves API almacenadas en el archivo .env se están cargando correctamente mediante python-dotenv.


Código

from dotenv import load_dotenv
import os

load_dotenv()

print(os.getenv("OPENAI_API_KEY"))
print(os.getenv("ANTHROPIC_API_KEY"))

Resultado esperado

Si la configuración es correcta, el script mostrará por pantalla las claves almacenadas en el archivo .env. 

sk-ope-xxxxxxxxxxxxxxxxxxxxxxxx
sk-ant-xxxxxxxxxxxxxxxxxxxxxxxx

Si aparece None, significa que la variable de entorno no ha sido encontrada o el archivo .env no se ha cargado correctamente.
