APIs

Antes de utilizar los modelos de OpenAI y Anthropic, es necesario obtener las claves API de cada proveedor y almacenarlas de forma segura.


Obtención de claves

Accede a los paneles de administración de:

  • OpenAI
  • Anthropic

Desde cada plataforma podrás generar una clave API asociada a tu cuenta.


Crear archivo .env


  └─$ nano .env

Dentro del archivo guarda las claves: 


  OPENAI_API_KEY=xxxx
  ANTHROPIC_API_KEY=xxxx

Sustituye xxxx por las claves generadas en cada plataforma.


Recomendaciones

  • No compartir el archivo .env.
  • No subir las claves a GitHub.
  • Utilizar siempre variables de entorno.
  • Proteger el archivo mediante permisos restrictivos.

  └─$ chmod 600 .env
