Uso Dual V1 (GPT + Claude)

Script base para trabajar con múltiples modelos de IA desde una única aplicación Python utilizando las APIs de OpenAI y Anthropic mediante variables de entorno.


Objetivo

  • Centralizar varios proveedores de IA.
  • Gestionar credenciales mediante variables de entorno.
  • Seleccionar el modelo desde terminal.
  • Unificar el flujo de consultas.

Funcionamiento

  1. Carga las claves desde .env.
  2. Inicializa los clientes OpenAI y Anthropic.
  3. Define una función para cada proveedor.
  4. Solicita el modelo a utilizar.
  5. Envía el prompt al proveedor seleccionado.
  6. Muestra la respuesta por terminal.

Código


    import os

    from dotenv import load_dotenv
    from openai import OpenAI
    from anthropic import Anthropic


    load_dotenv()


    openai_client = OpenAI(
        api_key=os.getenv("OPENAI_API_KEY")
    )

    anthropic_client = Anthropic(
        api_key=os.getenv("ANTHROPIC_API_KEY")
    )


    def ask_openai(prompt):

        response = openai_client.chat.completions.create(
            model="gpt-4.1-mini",
            messages=[
                {
                    "role": "user",
                    "content": prompt
                }
            ]
        )

        return response.choices[0].message.content


    def ask_claude(prompt):

        message = anthropic_client.messages.create(
            model="claude-sonnet-4-0",
            max_tokens=100,
            messages=[
                {
                    "role": "user",
                    "content": prompt
                }
            ]
        )

        return message.content[0].text


    model = input(
        "Elige IA (openai / claude): "
    ).strip().lower()

    prompt = input(
        "Escribe tu pregunta: "
    )


    if model == "openai":

        print("\nRespuesta OpenAI:\n")
        print(ask_openai(prompt))


        elif model == "claude":

            print("\nRespuesta Claude:\n")
            print(ask_claude(prompt))

        else:

            print(
                "Modelo no válido. Usa 'openai' o 'claude'."
            )
