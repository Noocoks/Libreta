Proyecto de IA local

python openai anthropic api venv

Apartado dedicado a la creación de un entorno local para trabajar con modelos de inteligencia artificial mediante APIs, automatizaciones y scripts desarrollados en Python.

El objetivo es disponer de una estructura organizada que permita probar, desarrollar y combinar distintos proveedores de IA desde un único entorno de trabajo.


Estructura recomendada

~/ai/
├── venv/
├── .env
├── .gitignore
├── test_"...".py

Creación del entorno virtual

└─$ mkdir ~/ai && cd ~/ai
└─$ python3 -m venv venv
└─$ source venv/bin/activate

Acceso rápido

└─$ cd ~/ai && source venv/bin/activate

Puedes utilizar cualquier ruta de trabajo. La mostrada aquí es la utilizada durante la documentación del proyecto.


Instalación de dependencias

└─$ pip install --upgrade \
openai \
anthropic \
shell-gpt \
ipython \
python-dotenv

Gestión de APIs

Las claves se almacenan dentro de un archivo .env para evitar exponerlas en código fuente o repositorios.

Permisos recomendados

└─$ chmod 600 .env

Esto restringe completamente el acceso al propietario del archivo.


Flujo de trabajo

  1. Crear el entorno virtual.
  2. Instalar dependencias.
  3. Configurar las APIs.
  4. Verificar el funcionamiento mediante scripts de prueba.
  5. Configurar Git y exclusiones del repositorio.
  6. Crear integraciones con OpenAI y Anthropic.
  7. Desarrollar aplicaciones que combinen múltiples modelos.
