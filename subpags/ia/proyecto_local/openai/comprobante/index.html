Script básico para verificar que la API de OpenAI está correctamente configurada y accesible desde Python.

El objetivo es validar la carga de variables de entorno, la autenticación y la comunicación con el modelo.


Dependencias

Código

📄 openai_tester.py
Python · 22 líneas 
import os

from dotenv import load_dotenv
from openai import OpenAI

load_dotenv()

client = OpenAI(
    api_key=os.getenv("OPENAI_API_KEY")
)

response = client.chat.completions.create(
    model="gpt-4.1-mini",
    messages=[
        {
            "role": "user",
            "content": "Hola"
        }
    ]
)

print(response.choices[0].message.content)
